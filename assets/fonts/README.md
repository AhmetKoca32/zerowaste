# Manrope font (asset)

Place the following TTF files here after downloading from [Google Fonts – Manrope](https://fonts.google.com/specimen/Manrope):

| Dosya            | Ağırlık (weight) |
|------------------|-------------------|
| Manrope-Light.ttf   | 300 (Book)  |
| Manrope-Regular.ttf | 400         |
| Manrope-Medium.ttf  | 500         |
| Manrope-Bold.ttf    | 700         |

1. Open the link above → "Download family".
2. Unzip and copy the four files into this folder (rename if your zip uses different names, e.g. `Manrope[wght].ttf` is variable; for static use the static TTF files from the zip).
